## 📁 项目结构

```
finance-software/
├── backend/           # NestJS 后端 (约60+ API)
├── frontend/          # Vue3 前端 (14个页面)
└── docs/              # 文档
```

---

## 🚀 运行说明

```bash
# 后端
cd backend && npm run start:dev
# API: http://localhost:3001/api

# 前端
cd frontend && npm run dev
# Web: http://localhost:3005
```
