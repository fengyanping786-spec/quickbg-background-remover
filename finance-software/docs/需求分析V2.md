## 1. 项目概述

### 1.1 项目背景

本系统参考柠檬云财税设计，为小微企业提供一套完整的财务核算解决方案。

### 1.2 核心特性

1. **多账套管理** - 支持多家公司，每家公司独立核算
2. **年度隔离** - 每个账套可按年份分开管理
3. **会计期间** - 支持月结、年终结账，反结账功能

### 2.2 凭证管理模块 (V2.0)

#### 2.2.1 功能列表

| 功能 | 描述 | 说明 |
|------|------|------|
| **新增凭证** | 借贷记账法录入 | 支持多行分录，自动平衡检查 |
| **查询凭证** | 多条件筛选 | 按日期、凭证号、摘要、金额等 |
| **批量操作** | 批量审核/删除/打印 | 勾选多条记录 |
| **凭证录入** | 智能分录 | 科目联想、金额自动计算 |
| **凭证审核** | 审核/反审核** | 支持批量审核 |
| **凭证过账** | 过账/反过账** | 月末结账前提 |
| **凭证模板** | 常用业务模板 | 差旅、工资、采购、销售 |
| **电子档案** | 凭证附件管理 | 图片/PDF上传查看 |
| **断号整理** | 整理凭证断号 | 自动填补空号 |

#### 2.2.2 凭证状态流转

```
草稿(draft) → 审核(approved) → 过账(posted)
     ↑             ↓              ↑
     └──── 反审核 ←┴─ 反过账 ←───┘
```

#### 2.2.3 科目体系

```
资产类 (1开头) - 借方增加
  1001 银行存款
  1002 库存现金
  1012 其他货币资金
  1122 应收账款
  1403 原材料
  1601 固定资产
  1602 累计折旧

负债类 (2开头) - 贷方增加
  2001 短期借款
  2201 应付票据
  2202 应付账款
  2221 应交税费
    -222101 增值税
    -222102 所得税

所有者权益 (4开头) - 贷方增加
  4001 实收资本
  4101 资本公积
  4103 利润分配
  4104 本年利润

成本类 (5开头) - 借方增加
  5001 生产成本
  5101 制造费用

损益类 (6开头)
  收入类(6001-6401) - 贷方增加
  费用类(6601-6801) - 借方增加
```

## 3. 技术架构

### 3.1 技术选型

| 层级 | 技术方案 |
|------|----------|
| 前端 | Vue3 + Element Plus |
| 后端 | NestJS + TypeORM |
| 数据库 | PostgreSQL (生产) / SQLite (开发) |
| 认证 | JWT |
| 文件存储 | 本地/云存储（预留） |

### 3.2 数据库设计要点

- 多账套：tenant_id 字段隔离
- 多年度：year 字段隔离
- 辅助核算：关联表设计

## 5. 功能优先级

### 第一阶段 (MVP)
- 凭证管理（增删改查、审核、过账）
- 账户管理
- 报表（资产负债表、利润表）
- 账套管理

### 第二阶段
- 账簿查询（明细账、总账）
- 期末结转
- 发票管理

### 第三阶段
- 税务申报
- 辅助核算
- 多币种
