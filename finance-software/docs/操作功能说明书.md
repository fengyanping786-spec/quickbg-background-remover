## 1. 系统概述

### 1.1 系统简介

本系统是一套专为小微企业设计的财务核算解决方案，支持多账套管理、会计期间管理、凭证录入、账簿查询、报表生成等核心财务功能。

### 1.2 主要功能模块

| 模块 | 功能 |
|------|------|
| 凭证管理 | 凭证录入、审核、过账、模板、附件 |
| 账簿查询 | 明细账、总账、日记账、多栏账 |
| 报表中心 | 资产负债表、利润表、现金流量表等 |
| 期末处理 | 结转、计提折旧、计提工资、月结年结 |
| 发票管理 | 发票录入、OCR识别、查重 |
| 税务管理 | 增值税计算、税负预警、所得税计算 |
| 账套管理 | 多公司账套、会计期间 |

### 1.3 登录系统

1. 打开系统登录页面
2. 输入用户名和密码
3. 点击"登录"进入系统

## 3. 凭证管理

### 3.1 新增凭证

1. 点击"凭证管理"菜单
2. 点击右上角"新增凭证"按钮
3. 填写凭证信息：
   - **凭证字**：选择凭证字（记/收/付/转）
   - **凭证号**：自动生成（可手动修改）
   - **凭证日期**：选择日期
   - **摘要**：输入凭证说明
4. 填写凭证分录：
   - 选择科目（输入科目编码或名称搜索）
   - 如需辅助核算，选择辅助核算类型和项目
   - 填写借方或贷方金额
5. 点击"添加分录"可继续添加
6. 系统自动检查借贷平衡
7. 点击"保存"完成

### 3.2 凭证审核

凭证保存后状态为"草稿"，需要审核才能过账：

1. 在凭证列表找到需要审核的凭证
2. 点击"审核"按钮
3. 凭证状态变为"已审核"

### 3.3 凭证过账

已审核的凭证可以过账：

1. 点击"过账"按钮
2. 凭证状态变为"已过账"
3. 过账后凭证参与账簿和报表汇总

### 3.4 批量操作

支持批量审核和删除：

1. 在凭证列表勾选要操作的凭证
2. 点击顶部出现的"批量审核"或"批量删除"按钮
3. 确认操作

### 3.5 凭证模板

系统提供常用凭证模板：

1. 点击"快捷凭证模板"展开
2. 点击模板卡片快速填充
3. 修改金额后保存

### 3.6 凭证附件

为凭证添加附件：

1. 打开凭证详情
2. 点击"上传附件"按钮
3. 选择图片或PDF文件上传

## 4. 账簿查询

### 4.1 明细账

查看指定科目的逐笔明细：

1. 点击"账簿查询"菜单
2. 选择"明细账"
3. 选择会计期间、科目
4. 查看明细分录

### 4.2 总账

查看科目汇总数据：

1. 选择"总账"
2. 选择年度和月份
3. 查看各科目本期发生额和余额

### 4.3 日记账

查看现金、银行存款日记账：

1. 选择"日记账"
2. 选择科目（1001银行存款、1002库存现金）
3. 查看逐笔收支记录

### 4.4 科目余额表

查看所有科目余额：

1. 选择"科目余额表"
2. 查看期初余额、本期发生、期末余额

## 5. 报表中心

### 5.1 资产负债表

查看企业资产、负债、所有者权益状况：

1. 点击"财务报表"菜单
2. 选择"资产负债表"
3. 选择年度和月份
4. 查看报表数据
5. 可导出Excel或CSV

### 5.2 利润表

查看企业经营成果：

1. 选择"利润表"
2. 选择年度和月份
3. 查看收入、成本、费用、利润

### 5.3 现金流量表

查看现金收支情况：

1. 选择"现金流量表"
2. 选择年度和月份
3. 查看经营活动、投资、筹资现金流

### 5.4 多期对比

对比不同期间报表数据：

1. 选择报表类型和期间
2. 在"对比期"下拉框选择要对比的年份
3. 查看对比数据

## 6. 期末处理

### 6.1 结转收入

将本期收入结转至本年利润：

1. 点击"期末处理"菜单
2. 选择"结转收入"
3. 点击"执行"
4. 系统自动生成结转凭证

### 6.2 结转成本

将本期成本结转至本年利润：

1. 选择"结转成本"
2. 点击"执行"

### 6.3 结转费用

将本期费用结转至本年利润：

1. 选择"结转费用"
2. 点击"执行"

### 6.4 计提折旧

计提固定资产折旧：

1. 选择"计提折旧"
2. 点击"执行"
3. 系统生成折旧凭证

### 6.5 计提工资

计提本月职工薪酬：

1. 选择"计提工资"
2. 点击"执行"
3. 系统生成工资凭证

### 6.6 月结

完成本月所有结转后，进行月结：

1. 点击"期间管理"
2. 选择对应月份
3. 点击"月结"

## 7. 发票管理

### 7.1 新增发票

1. 点击"发票管理"菜单
2. 点击"新增发票"
3. 填写发票信息（发票号码、类型、金额、税率等）
4. 保存

### 7.2 OCR识别

通过拍照或上传图片自动识别发票信息：

1. 点击"OCR识别"按钮
2. 选择"从URL识别"输入图片地址，或"选择图片"上传
3. 系统自动识别发票信息
4. 确认无误后"使用识别结果"

### 7.3 发票查重

系统自动检查重复发票：

1. 录入发票时
2. 如有重复，系统提示"该发票已存在"

### 7.4 关联凭证

将发票生成对应凭证：

1. 在发票列表点击"关联凭证"
2. 选择要关联的凭证

## 8. 税务管理

### 8.1 增值税计算

计算应纳税额：

1. 点击"税务管理"
2. 点击"计算增值税"
3. 填写销售额和进项税额
4. 系统计算应纳税额

### 8.2 税负预警

系统自动监控税负率：

- 税负率正常范围：1%-3%
- 低于1%：显示低风险预警
- 高于3%：显示高风险预警
- 预警信息显示在税务概览区域

### 8.3 所得税计算

预估企业所得税：

1. 点击"计算所得税"
2. 填写收入和成本费用
3. 系统计算应纳税额

### 8.4 申报表

查看和导出税务申报表：

1. 选择月份
2. 查看申报表数据
3. 可导出Excel

### 9.1 创建账套

1. 点击"账套管理"

2. 停用后该账套不可登录

### 常见问题

**Q: 凭证借贷不平衡怎么办？**
A: 检查分录金额，确保借方合计等于贷方合计。

**Q: 如何查看之前月份的报表？**
A: 在报表页面选择对应年度和月份即可。

**Q: 发票识别失败怎么办？**
A: 确保图片清晰，或手动录入发票信息。
