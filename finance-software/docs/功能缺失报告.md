## 三、建议处理

### 高优先级 (P1)

1. **凭证字关联** - 凭证新增/编辑时选择凭证字，生成完整凭证号（如：记-0001）
2. **多期对比** - 报表页面增加"对比"功能，选择期次对比

### 中优先级 (P2)

3. **OCR拍照** - 上传图片调用真实OCR API
4. **计提折旧** - 实现折旧计提功能
5. **税负预警** - 实现税负率监控和提醒
