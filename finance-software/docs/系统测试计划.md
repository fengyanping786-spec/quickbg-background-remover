### 1.2 测试类型

- **单元测试**：针对单个 Service/Controller 的功能测试
- **集成测试**：测试多个模块间的协作
- **E2E 测试**：从 API 到数据库的完整流程测试

## 4. 执行计划

### 4.1 本地执行

```bash
# 单元测试
npm run test

# 集成测试
npm run test:integration

# E2E 测试
npm run test:e2e
```

### 4.2 CI/CD 集成

- 每次 push 自动执行单元测试
- PR 合并前执行完整测试
- 测试覆盖率目标: 70%

## 5. 测试报告

测试报告生成命令：

```bash
npm run test:cov
```
