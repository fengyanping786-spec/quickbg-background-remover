# 🧠 记忆翻牌游戏

一个基于Web的记忆翻牌游戏，使用纯HTML、CSS和JavaScript实现。

## 🎮 游戏玩法

1. **目标**：翻开两张相同的牌消除它们
2. **时间限制**：60秒内完成所有配对
3. **计分规则**：
   - 每成功配对一次：+100分
   - 时间奖励：剩余时间 × 10分
   - 尝试次数越少，准确率越高

## 🛠️ 技术实现

### 1. CSS动画效果
- 3D翻转动画（`transform: rotateY()`）
- 平滑过渡效果（`transition: transform 0.5s`）
- 响应式网格布局（`grid-template-columns`）
- 渐变背景和阴影效果

### 2. JavaScript核心逻辑
- **数组随机化**：Fisher-Yates洗牌算法
- **游戏状态管理**：匹配检测、计时器、分数计算
- **事件处理**：点击、键盘快捷键、触摸支持
- **DOM操作**：动态创建卡片、更新状态显示

### 3. 游戏功能
- ✅ 计时器（60秒倒计时）
- ✅ 分数系统
- ✅ 尝试次数统计
- ✅ 提示功能（3次机会）
- ✅ 游戏结束弹窗
- ✅ 键盘快捷键支持
- ✅ 触摸屏优化
- ✅ 响应式设计

## 🎮 操作说明

### 鼠标/触摸操作
1. 点击"开始游戏"按钮
2. 点击卡片翻开
3. 翻开两张相同的卡片消除
4. 使用提示功能（点击提示按钮）

### 键盘快捷键
- **空格键** / **Enter键**：开始游戏
- **H键**：使用提示
- **R键**：重新开始
- **ESC键**：关闭弹窗

```javascript

// 游戏时间（秒）

// 提示次数
```
